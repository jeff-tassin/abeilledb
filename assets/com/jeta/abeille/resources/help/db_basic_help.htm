Basic View

The basic connection view assumes you are using the default JBDC drivers supplied by your database vendor.

Database Identifies the database you want to use for this connection. Abeille currently supports DB2, DaffodilDb, HSQLDB, JDataStore, McKOI, MySQL, Oracle, Pointbase and PostgreSQL
Name This is the name of the database instance. (For Oracle this is the SID and for McKOI this is the default schema).
Description An arbitrary description for this connection.
Server The hostname or IP address where your database is running.
Port The port number to use to connect to your database.
JDBC JAR File The JAR path/file(s) supplied by your database vendor.

*You don't have to supply the driver name in this view. Abeille assumes the default driver supplied by your vendor.
