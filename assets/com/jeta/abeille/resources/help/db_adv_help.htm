Advanced View

The advanced connection view allows you to specify any JDBC driver for the connection.

Database Identifies the database you want to use for this connection. Abeille currently supports DB2, DaffodilDb, HSQLDB, JDataStore, McKOI, MySQL, Oracle, Pointbase and PostgreSQL
Description An arbitrary description for this connection.
Driver The name of the JDBC driver class. The following table displays the default drivers for each supported database. If you are using a third party driver, then provide the name specified by your driver vendor.
URL The URL to use to connect to your database. The form of the URL is dependent on the database vendor. The following tables shows the default URLs for the supported databases
JDBC JAR File(s) The path/file(s) to your JDBC drivers.

The following table lists the default JDBC information for each database. However, in the advanced view, you can specify any JDBC driver that is compatible with your database. Keep in mind that Abeille uses PreparedStatements, so if you use a third party driver, make sure that is supports this capability.
